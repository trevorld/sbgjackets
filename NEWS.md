sbgjackets v0.1.0 (development)
===============================

New features
------------

Functions to help create small box game (SBG) jackets:

* Helper grob functions:

  + `creditsGrob()` (to go on inside of jacket)
  + `spineIconGrob()`
  + `spineTextGrob()`

* Other helper functions:

  + `check_sbgjackets_dependencies()` checks if this package's run-time system dependencies are installed (#19)
  + `pdf_polish_jacket()` adds instructions and/or metadata
  + `sbgjackets_style()` returns a marquee style set object (#18)

Functions to create small box game (SBG) jackets under Creative Commons licenses:

* Functions to create SBG jackets for dominoes (#1):

  + `sbgj_dominoes_double6()`
  + `sbgj_dominoes_double9()`
  + `sbgj_dominoes_double12()`
  + `sbgj_dominoes_all()`

* Functions to create SBG jackets for generic components (#8):

  + `sbgj_black_stones()`
  + `sbgj_dice()`
  + `sbgj_glass_stones()`
  + `sbgj_marbles()`
  + `sbgj_pawns()`
  + `sbgj_polyhedral_dice()`
  + `sbgj_reversible_discs()`
  + `sbgj_white_stones()`

* Functions to create Playing Card Box jackets:

  + `pcbj_bavarian_pattern()`
  + `pcbj_bridge()`
  + `pcbj_castilian_pattern()`
  + `pcbj_english_pattern()`
  + `pcbj_decktet()`
  + `pcbj_everdeck()`
  + `pcbj_mahjong()`
  + `pcbj_pinochle()`
  + `pcbj_poker()`

Functions to create small box game (SBG) jackets for **Personal Use Only**:

* Functions to create SBG jackets for GIPF games (#11):

  + `sbgj_yinsh()`

* Functions to create SBG jackets for Looney Pyramids (#2):

  + `sbgj_looney_pyramids()`
  + `sbgj_homeworlds()`
  + `sbgj_ice_duo()`
  + `sbgj_jinxx()`
  + `sbgj_martian_chess()`
  + `sbgj_nomids()`

* Functions to create SBG jackets for nestorgames (#3):

  + `sbgj_shibumi()`

* Functions to create Playing Card Box jackets for Foxtrot Games:

  + `pcbj_fox_in_the_forest()`

* Functions to create Playing Card Box jackets for *Wizard*:

  + `pcbj_wizard()`
